\section{Stakeholders}
Individuals who may be interested in the system include:
\begin{itemize}
  \pbodyitem{Consumers}{
    Individuals who view or request graphic design produced by the system.
    They are interested in obtaining high-quality prints or images.
  }
  \pbodyitem{Implementers}{
    Developers responsible for building the system.
    They are interested in building easy-to-extend software.
  }
  \pbodyitem{Users}{
    Individuals produce graphic designs using the system.
    They are interested in being able to easily use the system.
  }
\end{itemize}
